#Installation

Magento SSO module installation is very easy, please follow the steps for installation-

1. Make sure compiler is disabled.

2. Unzip the respective magento_sso_login.zip and then move "app" and "lib" folders (inside "src" folder) into magento root directory and flush cache and reindex all from admin panel.
